Com o média query vamos criar configuraçôes adequadas para cada tela, vamos inserir alguns itens para os testes.

Lá em cima do link css (Link rel="stylesheet" href="style.css") vamos inserir o média depois do href (link rel="stylesheet" href="style.css" média="")

Temos alguns tipos de media.

media="all" para todos os tipos.

media="screen" para telas.

media="print" para impressão

em screen temos ainda algumas configurações

media="screen and (orientation:landscape)" para paginas na horizontal ou paisagem

media="screen and (orientation:portatil)" para paginas na vertical ou retrato

Podemos tabem fazer outras configurações mais personalizadas de acordo com esta tabela:

telas pequenas ate 600px.

celular de 600px ate 768

tablet de 768 ate 992px

desktop 992px ate 1200px

telas acima de 1200px


ultilizamos o media de acordo com a pagina :

@media screen and (win-width:768px)
