Links e Âncoras externos

Para utilizarmos os links ou ancoras nos utilizamos a tag a href=""

Exemplo:

O site do Google é um site de referência de busca

Embora ele seja referência mas existem outros dispositivos de busca que podem trazer resultados diferentes

Como os exemplos abaixo:
Yahoo,
Msn,
Bing,
Duckduckgo,
Ecosia entre outros.

Agora nos vamos clicar nos links sem sair da página

É importante inserirmos algumas tags se não o usuário pode não voltar mais na nossa página.

Para isso vamos abrir outra aba sem sair da nossa página, vamos utilizar os mesmos links acima, mas com as tags target="_blank" rel="external" junto com o hiperlink.

_blank vai abrir o link em uma janela em branco.
_self vai abrir o link na janela ou guia atual (padrão)
